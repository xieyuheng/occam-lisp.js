# occam-lisp.js

A lisp without unnecessary entities.

## Usages

### Command line tool

Install it by the following command:

```sh
npm install -g @xieyuheng/occam-lisp.js
```

The command-line program is called `occam-lisp.js`.

## Development

```sh
npm install
npm run build
npm run test
```
